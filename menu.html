Les langages de formations

HTML CSS
Langage de balisage utilisé pour structurer les pages web.
Il permet d'afficher du texte, des images, des vidéos et d'autres
Langage de mise en forme utilisé pour styliser les pages HTML.
Il permet de définir les couleurs, politiques, tailles, animations
et mises en page des sites web.

JAVASCRIPT
Langage de programmation utilisé pour ajouter de l'interactivité aux pages web.
Il fonctionne côté client (navigateur), mais aussi côté serveur avec Node.js .
Indispensable pour le développement d'applications modernes (React, Vue, Angular).

PHP
PHP (Hypertext Preprocessor) Langage de programmation côté serveur utilisé
pour créer des sites dynamiques. Il permet de gérer des bases de données,
des sessions et des formulaires. PHP est souvent utilisé avec MySQL pour
stocker et manipuler des données

DARK
Langage développé par Google, principalement utilisé avec le framework Flutter .
Il est conçu pour le développement d'applications mobiles, web et desktop.
Dart est un langage orienté objet et compilé en code natif, ce qui le rend
très performant.

JAVA
Langage de programmation orienté objet utilisé pour le développement
d'applications web, mobiles et desktop.Très populaire dans le
développement Android (via Android Studio).Il est portable, robuste
et utilisé dans les grandes entreprises pour des applications complexes.

fromwork

BOOTSTRAP
Framework CSS développé par Twitter pour faciliter la conception des
sites web.Il propose un système de grille responsive, des composants UI
(boutons, cartes, alertes).Très utilisé pour créer rapidement des interfaces
modernes et adaptables à tous les écrans.

Flutter
Framework développé par Google pour créer des applications mobiles,
web et desktop avec une seule base de code. Utilisez le langage Dart
et le moteur graphique Skia pour un rendu rapide et fluide.
Très populaire pour son design moderne , ses animations fluides
et ses performances élevées.

React JS
Bibliothèque JavaScript développée par Facebook (Meta) pour créer des
interfaces utilisateur dynamiques.Basé sur un concept de composants
réutilisables et un rendu rapide avec le Virtual DOM .Utilisé pour le
développement d'applications web modernes (ex: Facebook, Instagram).

React Native
Framework basé sur React JS permettant de créer des applications mobiles
pour Android et iOS .Utiliser le langage JavaScript pour générer du code natif.
Permet un développement plus rapide et un partage de code entre les plateformes.

Laravel
Framework PHP open-source basé sur le modèle MVC (Modèle-Vue-Contrôleur) .
Facilite le développement d'applications web robustes avec une gestion
simplifiée des routes, migrations et bases de données. Propose des fonctionnalités
avancées comme Eloquent ORM , Blade Templating , et Authentification intégrée .
